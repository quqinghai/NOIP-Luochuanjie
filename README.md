# 命名规则  
***
本项目中有来来自三个题库的代码，分别时  
NOI题库 : [http://noi.openjudge,cn](http://noi.openjudge.cn)  
CodeVS : [https://codevs.cn](https://codevs.cn)  
洛谷 : [https://www.luogu.com](https://www.luogu.com)  

### NOI题库命名规则
> x.y.z.cpp

其中 `x.y` 表示NOI题库第x.y节
`z` 表示在该节中的题号

### CodeVS命名规则
> codevs.x.cpp 

其中 `x` 表示本题在CodeVS中的题号

### 洛谷命名规则
> luogu.x.cpp 

其中 `x` 表示本题在洛谷中的题号
